Este projeto tem como finalidade criar um aplicativo de relacionamento (dating app) com Angular + ASPNET, com as features que esses aplicativos geralmente possuem, como matches, mensagens, edição de perfil, etc.


A decisão do tema do projeto ser um aplicativo de relacionamento é devido a um app assim possuir várias features diferentes, assim há mais assets semelhantes ao que uma empresa pode requerir de um desenvolvedor.


Você também pode acessar o meu Github para clonar o projeto e executá-lo na sua máquina, clique aqui para acessar o repositório


Features existentes no aplicativo:


👉 Autenticação via JWT (JSON Web Token);

👉 Upload e armazenanento de fotos via Cloudinary;

👉 Mensagens em tempo real via SignalR;

👉 Likes em tempo real via SignalR.


Features que pretendo implementar no aplicativo:


👉 Senha esquecida;

👉 Tradução adequada para o português brasileiro.
